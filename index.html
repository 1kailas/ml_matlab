Overview

Project Overview

This project demonstrates a full MATLAB Machine Learning pipeline using a diabetes dataset. It includes preprocessing, training a Support Vector Machine, predictions, accuracy evaluation, and confusion matrix visualization.

Step 1

Load Dataset

dataset = readtable("Diabetes Missing Data.csv");
Step 2

Data Cleaning

glucose_avg = round(mean(dataset.Glucose,"omitmissing"));
dataset.Glucose(ismissing(dataset.Glucose)) = glucose_avg;
Step 3

Train-Test Split

data_partition = cvpartition(size(dataset,1),"HoldOut",0.3);

training_dataset = dataset(training(data_partition),:);
testing_dataset = dataset(test(data_partition),:);
Step 4

Model Training

predictor = training_dataset(:,1:end-1);
response = training_dataset(:,end);

trained_mdl = fitcsvm(predictor, response);
Step 5

Prediction

predictor_test = testing_dataset(:,1:end-1);
response_test = testing_dataset(:,end);

pred = predict(trained_mdl, predictor_test);
Step 6

Evaluation Metrics

correct_pred = nnz(pred == response_test.Class);
wrong_pred = nnz(pred ~= response_test.Class);

accry = (correct_pred / (correct_pred + wrong_pred)) * 100;

📊 Confusion Matrix

confusionchart(response_test.Class, pred);

The confusion matrix shows correct vs incorrect classifications visually.
